How to build ?

  Download .NET 10 and git. . .
  
  Using the git clone this repository. . .
  
  Copy Assembly-CSharp.dll from your ULTRAKILL files to a new folder called lib. . .
  
  Run dotnet build. . .
  

  Move the bitmotte.LessScaryMannequins.dll file in bin/Debug/netstandard2.1 to your mods folder. . .
  
  Done ! ! !
